Tool or Language Link to Doc Description Example(s) of Syntax
Git Git-scm Copy/clone files to local: 
$git clone <repository_link>
Create a new repository: 
$git init
Create a branch: 
$git branch <branch_name>
Staging a commit: $git add <file_path>
Commit changes: $git commit -m "A commit message"
Push to GitHub: $git push
See changes $git status
See changes that will go into next commit (more detailed than $git status) $git diff --staged
SVN SVN book Copy/clone file: $svn checkout
Create a new reository: $svn create
wGet GNU wGet Download file using URL: $wget <URL>
wget -O flag to rename file when dl'ing: $wget -O <new_file_name> <URL>
CURL CURL doc Download file using URL: $curl -O <URL>
CURL -o flag to rename file when dl'ing: $curl -o <new_file_name> <URL>
AWK GNU Awk Sort file.csv alphabetically by 1st column to sorted.csv: $awk '{print | "sort --field-separator=',' -k1}' file.csv > sorted.csv
Sort file.csv alphabetically by 1st column to newFile.csv: $awk '{print | "sort --field-separator=',' -k1}' file.csv > newFile.csv
Print all rows where text in column 1 begins with M: $awk '/^M/ {print}' file.csv > newFile.csv
Print only column 1 where text in column 1 begins with M: $awk '/^M/ {print $1}' file.csv > newFile.csv
Say you want to print col1&2 where column 2 are int and you want to divide it by 2000: $awk '/^M/ {print $1, $2/2000}' file.csv > newFile.csv
Make GNU Make Using Makefile to curl file.csv and execute awkFile.awk
default:
# Download file.csv using CURL:
curl -O https://example.com/file.csv

# Make awkFile.awk executable:
chmod +x awkFile.awk

# Run awkfile.awk on file.csv and put ouptut into newFile.csv
awk -f awkFile.awk file.csv > newFile.csv
